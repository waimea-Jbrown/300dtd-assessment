Some Definitions

HTML

HTML stands for Hyper-Text Markup Language

Web browsers read HTML and follow the code to render a web page. HTML contains the content of the page: the text, the page structure, etc.

CSS

CSS stands for Cascading Style Sheets

Web browsers use CSS to alter the styling of a web page. The styling is kept seperate from the HTML content to make updating things easier.

JS

JS stands for JavaScript

Web browsers use JS to make pages interactive. JS code can be used to modify the page as the user interacts with it.
